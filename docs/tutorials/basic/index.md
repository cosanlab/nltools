# Basic Tutorials

Welcome to the nltools basic tutorials! These tutorials are designed to get you started quickly with the three main classes in nltools.

## Getting Started

The tutorials in this section cover the essential functionality you need to begin using nltools:

1. **[Brain_Data Basics](01_brain_data_basics.ipynb)** - Learn how to load, manipulate, and visualize neuroimaging data
2. **[Design_Matrix Basics](02_design_matrix_basics.ipynb)** - Create experimental designs for fMRI analyses
3. **[Adjacency Basics](03_adjacency_basics.ipynb)** - Work with similarity and distance matrices
4. **[Basic Analysis Workflow](04_basic_analysis_workflow.ipynb)** - Combine the classes in a complete analysis

## Prerequisites

Before starting these tutorials, make sure you have:
- nltools installed (`pip install nltools`)
- Basic familiarity with Python and NumPy
- Understanding of fMRI data structure (optional but helpful)

## Quick Tips

- Each tutorial is self-contained and can be run independently
- The tutorials use minimal example data to keep things simple
- Focus is on core functionality - advanced features are covered in other sections
- All code examples are designed to run quickly on a standard laptop
