# `nltools.simulator`

**Simulated Neuroimaging Data**

Generate synthetic fMRI data for testing and benchmarking analyses. Useful for validating pipelines, testing statistical methods, and creating reproducible examples.

- `Simulator` — Generate synthetic brain data with controlled signal and noise
- `SimulateGrid` — Parameter grid for systematic simulation studies
