# `nltools.algorithms.inference`

GPU-accelerated statistical inference for neuroimaging data.

## Overview

The `nltools.algorithms.inference` module provides fast permutation testing and bootstrap resampling with optional GPU acceleration via PyTorch. When GPU is unavailable, efficiently uses CPU parallelization.
